                                            Access modifier in python 

                            access modifiers are used to set of limit of member accessibility


why we use access modifier ?

1> Access modifiers are used by object oriented programming languages like C++,java,python etc

2> Security

3> Maintainability

4> Code Clarity 


###########                    There are three type                    ########################

 1> public

 2> protected 

 3> privete 


public :- they are accessible from anywhere in the program

Protected members − They are accessible from within the class as well as by classes derived from that class.

Private members − They can be accessed from within the class only.


for example : phone call  red button or green button
